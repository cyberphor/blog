## Environment variables
```bas
echo $PATH

mkdir ~/scripts
vim ~/scripts/ping_google

# add to script
ping 8.8.8.8 | awk '{print $5,$7}'

chmod +x ~/scripts/ping_google
export PATH="$PATH:~/scripts"
source ~/.profile

echo $PATH
ping_google
```

## Permissions
```bash
vim hello_world.py

# add to source file
#!/usr/bin/python3
```
