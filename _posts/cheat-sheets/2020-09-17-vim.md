## Create file
```bash
# use Vim to create a new file
vim file01.txt

# change into 'insert' mode
i

# insert data
hello world!

# escape from 'insert' mode
[esc]

# write your changes to disk and quit
:wq
```

## Navigate
```bash
# enter 'Normal' mode
[esc]

# end of file
G

# end of line
$

# start of line
0

# start of file
gg

# search for the string 'life'
/life
```
